=-=-=-=-=-=- INSTRUÇÕES ANOTADAS DA AULA =-=-=-=-=-=-
- Na obtenção dos dados, se TXT/CSV, ideal utilizarmos a decodificação UTF-8 em caso deles virem bugados. Se não vierem bugados, ok manter, pelo que entendi.
- Veja delimitador, é o que separa os dados em linhas e colunas. PowerBI tende a construir automatico.
- Findada as configurações, antes de importar a base, podemos Carregar, Tranformar Dados ou Cancelar. Ideal clicarmos para TRANSFORMAR DADOS (Etapa T do ETL), seremos enviados ao PowerQuery.
- Imediatamente somos apresentados à linguagem M (que é o que aparece na barra de edição dos dados)


== Diferença de importar PLANILHAS GOOGLE de WEB ==
- Importamos de forma WEB as planilhas diversas (Google ou outras) que não são de NOSSA autoria. Ou seja, que já estão publicadas na WEB.
- Utilizamos Planilhas Googles apenas quando SOMOS PROPRIETÁRIOS da planilha.

== Linguagem M e Linguagem DAX ==
- Linguagem M usa-se no powerQuery;
- DAX usa-se quando abrimos o menu de exibição de tabelas ao lado esquerdo.
