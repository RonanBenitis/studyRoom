ATALHOS PARA O VS CODE

Alt+Z faz com que o código seja mostrado dentro da exibição, ou seja, que não ultrapasse a janela. Note conforme o texto ou o código desce quando o Alt+Z está ativado, a sua linha aumenta, ou seja, mesmo que na exibição aparece, no código não é acrescentado mais linhas.
